# Feature 02. Карточка кулинарного класса

## Цель задачи

Реализовать подробную карточку выбранного кулинарного класса.

Пользователь должен иметь возможность нажать кнопку «Подробнее» в списке классов и открыть экран с детальной информацией о занятии.

---

## Связанные требования

- FR-006 — приложение должно отображать подробную информацию о выбранном классе.
- FR-007 — карточка класса должна содержать программу занятия.
- FR-008 — карточка класса должна показывать данные о шефе.
- FR-009 — приложение должно показывать свободные места, цену, дату и адрес занятия.
- FR-019 — клиент должен видеть информацию о фартуке и наборе ножей, если эта опция включена в MVP.

---

## Что было реализовано

- Добавлен контейнер `classDetails` в `index.html`.
- В mock-данные добавлены подробные поля:
    - полное описание класса;
    - меню занятия;
    - описание шефа;
    - адрес;
    - количество комплектов студии;
    - стоимость комплекта студии.
- В `api.js` добавлен метод `getClassSlotById(slotId)`.
- В `ui.js` добавлена функция `renderClassDetails(slot)`.
- Добавлено переключение между списком классов и подробной карточкой.
- Добавлена кнопка «Назад к списку».
- Добавлена обработка ошибки, если класс не найден.
- Добавлены стили для подробной карточки класса.

---

## Использованные промпты

```text
Реализуй вторую фичу для клиентского приложения кулинарной студии «Шеф-стол»: подробную карточку выбранного класса.

Нужно:
- по клику на кнопку «Подробнее» открывать карточку выбранного класса;
- получать данные через mock API методом getClassSlotById(slotId);
- показывать фото, название, полное описание, меню, шефа, дату, длительность, адрес, цену и свободные места;
- показать информацию о фартуке и наборе ножей;
- добавить кнопку «Назад к списку»;
- не смешивать всю логику в одном файле;
- изменения внести в существующую структуру: api.js, ui.js, app.js, mock-data.js и style.css.
```

## Ручная проверка

Проверка выполнялась через открытие файла `src/index.html` в браузере.

Проверено:

- список кулинарных классов загружается;
- при нажатии на кнопку «Подробнее» открывается подробная карточка выбранного класса;
- в карточке отображаются фото, название, полное описание, меню, шеф, дата, длительность, адрес, цена и свободные места;
- отображается информация о фартуке и наборе ножей;
- кнопка «Назад к списку» возвращает пользователя к списку классов;
- у класса без свободных мест кнопка в списке остаётся неактивной;
- подробная карточка корректно отображается в мобильном размере браузера.
