# Feature 01. Список кулинарных классов

## Цель задачи

Реализовать главный экран клиентского приложения кулинарной студии «Шеф-стол», на котором клиент может увидеть список ближайших кулинарных классов.

Экран должен показывать основные данные по каждому классу: название, описание, шефа, дату, длительность, стоимость и количество свободных мест.

---

## Связанные требования

- FR-001 — приложение должно отображать список доступных кулинарных классов.
- FR-002 — карточка класса должна содержать основные сведения о занятии.
- FR-003 — приложение должно показывать количество свободных мест.
- FR-004 — если список классов пустой, должно отображаться понятное пустое состояние.
- NFR-006 — интерфейс должен быть адаптирован под мобильный экран.

---

## Что было реализовано

- Создана структура клиентской части в папке `src`.
- Добавлен главный HTML-файл `src/index.html`.
- Стили вынесены в `src/css/style.css`.
- JavaScript-код разделён на несколько файлов:
    - `mock-data.js` — тестовые данные;
    - `api.js` — mock API;
    - `utils.js` — форматирование даты, цены и текста свободных мест;
    - `ui.js` — отрисовка интерфейса;
    - `app.js` — запуск приложения.
- Добавлены карточки кулинарных классов.
- Добавлены фотографии блюд в карточки.
- Реализовано отображение статуса свободных мест.
- Реализована корректная форма слова «место»: `1 место`, `3 места`, `5 мест`.
- Добавлено состояние загрузки расписания.
- Добавлено пустое состояние для случая, когда список классов пустой.
