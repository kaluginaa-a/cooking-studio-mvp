# Bug 02. Ошибка загрузки расписания после добавления формы записи

## Симптом

После добавления формы записи приложение перестало показывать список кулинарных классов.

Вместо карточек на экране отображалось сообщение:

```text
Не удалось загрузить расписание. Попробуйте позже.
```

## Связанное требование

- FR-001 — приложение должно отображать список ближайших кулинарных классов.
- FR-002 — список классов должен загружаться из mock API.
- FR-037 — приложение должно показывать понятное сообщение при ошибке загрузки данных.

## Как воспроизвести

1. Открыть `src/index.html` в браузере.
2. Дождаться загрузки приложения.
3. Проверить область списка кулинарных классов.

## Фактический результат

Список классов не отображался.  
Пользователь видел сообщение об ошибке загрузки расписания.

## Ожидаемый результат

После открытия приложения должен отображаться список ближайших кулинарных классов из mock-данных.

## Причина

В файле `src/js/api.js` была синтаксическая ошибка: после метода `getClassSlotById(slotId)` не хватало запятой перед методом `createBooking(payload)`.

Из-за этого объект `api` не создавался корректно, метод `api.getClassSlots()` не выполнялся, а приложение попадало в обработчик ошибки.

## Исправление

В `src/js/api.js` была исправлена структура объекта `api`.

После метода `getClassSlotById(slotId)` была добавлена запятая:

```js
  },

  createBooking(payload) {
```

Также была проверена логика обработки клика в `src/js/app.js`, чтобы кнопки подробной карточки работали корректно.

## Использованный промпт

```text
Проверь, почему после добавления формы записи приложение показывает ошибку загрузки расписания. Найди синтаксическую или логическую ошибку в JS-файлах и предложи минимальное исправление.
```

## Ручная проверка

Проверка выполнялась через открытие файла `src/index.html` в браузере.

Проверено:

- приложение открывается без ошибки загрузки;
- список кулинарных классов отображается;
- карточки классов загружаются из mock-данных;
- кнопка «Подробнее» открывает подробную карточку;
- кнопка «Записаться» открывает форму записи;
- ошибка «Не удалось загрузить расписание» больше не появляется.

## Результат

Баг исправлен.  
Расписание снова загружается корректно.
